Coding Challenges

Here in this section, I would like to present you with some coding challenges that I have created along my learning journey. Highlighting my JavaScript skills.

Zedred App
Zedred App
The Zedred Challenge

The Zedred App is a simple app that let's the user play with some numbers. It is more like a JavaScript coding exercise. Using loops, functions, if/then/else statements, and some logic. The user enters two numbers between 0 and 100 or more. The app will check if the numbers are even, and will display them in BOLD.

Nasa App
Nasa App
The Nasa Challenge

A JavaScript coding exercise using APIs. The program fetchs data from an API. In this case we will use Nasa RestfulAPIs. The user can retrieve photos taken on Mars, and get a brief description of what camera was used, the date when the rover landed, and the date on earth. .

FizzBuzz App
FizzBuzz App
The FizzBuzz Challenge

The FizzBuzz App is a simple app. That checks if a number is multiple of 3 or 5 or both. If the number is just multiple of 3 then the program displays Fizz, if the number is multiple of 5 the program displays Buzz. In the case of a number multiple of 3, and 5 the program will display FizzBuzz with orange background.
